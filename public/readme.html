From the file sizes we can see that bootstrap.css is quite large. At 145,896 bytes, bootstrap.css is almost 29 times larger than style.css, which was the style sheet used for our vanilla soundboard. Also we can see that the Bootstrap version has very poor coverage compared to the vanilla version, specifically looking at the amount of unused bytes. The majority of unused bytes from the vanilla versions is from white space, whereas the majority of unused bytes for the Bootstrap version is actually unused CSS.
